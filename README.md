# Task Manager (Express + EJS)

A task management application built with Node.js, Express, and EJS that allows users to create, view, and manage tasks with title and description.

## Features
- Add tasks with title and description
- View all tasks
- File-based storage
- Simple UI using EJS templates

## Tech Stack
- Node.js
- Express.js
- EJS
- File System (fs module)

## How to Run Locally

1. Clone the repository (git clone https://github.com/redriot10/task-manager-express-ejs.git)
2. Install dependencies
3. Start the server
4. Open in browser
